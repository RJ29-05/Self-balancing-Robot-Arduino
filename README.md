# Self-balancing-Robot-Arduino
A two-wheeled self-balancing robot built using an MPU6050 sensor, Arduino microcontroller, L298N motor driver and PID control algorithm to achieve real-time balance and stability.
This project focuses on the design and development of a two-wheeled self-balancing robot using embedded systems and control engineering concepts. The robot uses an MPU6050 Gyro sensor to continuously measure its tilt angle and angular velocity, processes the sensor data through a microcontroller, and applies a PID control algorithm to adjust motor speed and maintain balance.

The system integrates hardware and software components including motor drivers, DC motors, sensors, and embedded firmware to achieve real-time stabilization. The project explores concepts such as sensor calibration, feedback control systems, PWM-based motor control, and real-time data processing.

Key Features:

Real-time self-balancing using PID control
MPU6050 calibration
Closed-loop motor speed control
PWM motor driving
Embedded C firmware implementation
Sensor calibration and filtering

Technologies Used:

Microcontroller: Arduino UNO
Programming: Embedded C
Sensors: MPU6050
Motor Driver: L298N
Control Algorithm: PID Controller

This project demonstrates the application of embedded systems, robotics, and control theory to develop an autonomous balancing system.
